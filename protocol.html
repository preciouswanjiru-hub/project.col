Transmission Control Protocol/Internet Protocol

This is the foundational communications framework that enables web development by standardizing how data is exchanged via the internet. It has 4 layers.

Application Layer

This is where the web applications interact with the network. Consists of the high-level protocols like HTTP, FTP, and SMTP.

Transport Layer

Ensures reliable and efficient delivery of data between devices. It performs:

  1. Segmentation and reassembly- Breaks data into smaller packets for transmission and reassembles them at the destination.
  2. Connection management- Establishes, maintains and terminates connections between devices.
  3. Flow control- Manages the rate of data transmission to prevent congestion.
  4. Error detection and correction- Uses checksums to detect errors in transmitted data and requests retransmission if necessary.
TCP- Used when reliablitity and accuracy are essential. UDP- Used when speed is more essential than perfect accuracy.

Internet Layer

Responsible for addressing, packaging and routing of data packets. It performs:

  1. Logical addressing- Assigns unique IP addresses to devices on the network.
  2. Packet routing- Determines the best path for data packets to reach their destination.
  3. Fragmentation and reassembly- Breaks large packets into smaller fragments for transmission and reassembles them at the destination.
  4. Protocol support- Provides services for other protocols to use.

Network Access Layer

Responsible for the physical transmission of data over network hardware. It performs:

  1. Physical addressing- Uses MAC addresses to identify devices on the local network.
  2. Data framing- Encapsulates data packets into frames for transmission over the physical medium.
  3. Error detection- Uses techniques like checksums to detect errors in transmitted frames.
  4. Media access control- Manages access to the physical medium to prevent collisions and ensure efficient use of the network.
  5. Access control- Ensures only authorized devices can access the network.
