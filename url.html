Uniform Resource Locator (URL)

This is a unique identifier that specifies the location of a resource on the internet. It is contained by all the resources available on the internet. It is also known as web address. An example is: https//www.geeksforgeeks.org/

Different parts of a URL

A URL consists of several components that provide information about the resource being accessed.

Reference URL

https://www.geeksforgeeks.org/dsa/array-data-structure-guide/#what-is-array

The different parts of this URL are:

  1. Scheme- The protocol used to access the resource which is https
  2. Domain- The server where the resource is located which is www.geeksforgeeks.org
  3. Port- The communication endpoint used to access the resource which is 80 It is not visible in the URL because it is the default port for HTTP protocol
  4. Path to the file- The location of the resource on the server which is /dsa/array-data-structure-guide/
  5. Query- The parameters passed to the server which is empty in this URL. Usually ?
  6. Parameters- Additional information passed to the server such as ref= home-articlecards
  7. Fragment- The specific part of the resource which is #what-is-array
