Uniform Resource Locator (URL)

This is a unique identifier that specifies the location of a resource on the internet. It is contained by all the resources available on the internet. It is also known as web address. An example is: https//www.geeksforgeeks.org/

Different parts of a URL

A URL consists of several components that provide information about the resource being accessed.
