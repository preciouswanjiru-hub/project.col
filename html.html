Hypertext Markup Language (HTML)

This is a code that tells browsers how to display a webpage such as text, media and links. It is a static language thus it does not support dynamic content meaning it does not inherently provide interactive features but can be combined with CSS and JavaScript for interactivity.

Basic Structure of an HTML Document

  4. Body Element- This section contains the actual content of the webpage, such as text, images and links. It is represented by the tag.
